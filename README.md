# Ленточки на иероглифе: топология поверхностей

Проект по Python (Phys&Math).  
Интерактивное визуальное моделирование поверхностей, получаемых приклеиванием перекрученных/неперекрученных ленточек между краевыми окружностями иероглифа, с вычислением гомеоморфного типа (сфера с `g` ручками / `h` дырками / `m` пленками Мёбиуса).

## Идея проекта
 
Пользователь может:

- Выбирать две краевые окружности и приклеивать между ними **ленточку** (прямоугольную полоску).
- Задавать для каждой ленточки **перекрученность** (0 — без перекручивания, 1 — с одним перекручиванием, в перспективе — произвольное целое число).
- **Перемещать** точки крепления ленточки вдоль окружностей (менять углы).
- **Менять порядок слоёв** (какая ленточка выше/ниже при отрисовке).
- В реальном времени видеть в углу экрана **число ручек** (`g`) , **число дырок** (`h`), **число пленок Мёбиуса** (`m`), т.е. род поверхности, гомеоморфной получившейся конструкции.
